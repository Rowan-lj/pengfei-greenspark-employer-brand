## 零、鹏飞校招真实画像

以下数据来自鹏飞 2026 年校园招聘简章、各高校就业网及第三方求职平台搜索。

### 0.1 基本盘

| 维度 | 数据 | 对方案的约束 |
|------|------|------------|
| 招聘规模 | 约 50 人 | 走精准深度路线，不做广撒网 |
| 薪资 | 6,000-9,000 元/月 | 不拼数字，拼成长速度和购买力 |
| 工作地点 | 山西孝义（总部）+ 晋城、长治、阳泉、天津、十堰 | 翻转叙事：小地方=大空间，薪资购买力×3 |
| 投递方式 | 仅邮箱，无自动确认 | 方案必须包含投递体验改造 |
| 宣讲会 | 省内 3-5 所高校，省外无专场 | 线上内容是唯一省外触达路径 |
| 社媒存在感 | 小红书/B站/牛客搜索结果为零 | 从零建内容，而非优化已有内容 |
| 校招项目 | 鸿鹏生（普通）+ 麒鹏生（管培，985/211门槛） | 麒鹏生门槛-薪资落差须主动解释 |
| 核心资产 | 氢能链主、2万吨制氢已投产、氢能重卡已上路 | 真实产业资产，从未被转化为传播内容 |

### 0.2 能源行业竞争格局

| 梯队 | 代表 | 规模 | 薪资 | 学生认知 | 鹏飞的位置 |
|------|------|:--:|------|------|------|
| 央企巨头 | 国家能源、中石化、中石油 | 万人级 | 硕 15-25W | 稳定、铁饭碗 | 拼不了稳定，拼可见度 |
| 新能源热门 | 宁德时代、比亚迪、隆基 | 数千人 | 硕 18-33W | 高薪、风口 | 薪资差 2-4×，拼成长速度 |
| 区域强企 | 特变电工 | ~1,700人 | — | 装备中国、年轻化 | 鹏飞最直接的对标对象 |
| **鹏飞** | — | **50人** | **6-9K** | **零认知** | 氢能链主是真实差异化 |

### 0.3 鹏飞的三个约束和三个优势

**约束**：

1. **薪资天花板**。6-9K 在 2026 年校招市场没有数字竞争力。应对：重新定义比较框架——不对标绝对薪资，对标薪资÷生活成本的实际购买力。
2. **地域引力不足**。孝义是县级市。应对：翻转叙事——「你在北京写字楼是第 500 号员工，在鹏飞制氢基地是前 50 个定义氢能的工程师」。
3. **规模小，无法制造 FOMO**。50 人无法靠从众效应驱动投递。应对：把小变成精——「我们只要 50 人，因为为每人匹配一位技术导师需要投入」。

**优势**：

1. **氢能链主的产业稀缺性**。2026 年中国真正跑通氢能全产业链的民营企业不超过 5 家。这个事实从未被转化为面向学生的故事。
2. **小平台=快速被看见**。50 人的校招批次意味着入职第一年管理层就知道你的名字。在比亚迪 3.18 万人中你是一个工号，在鹏飞你是「2026 届那批人」。
3. **零社媒存在=一张白纸**。鹏飞不像宁德时代和比亚迪已有固化的口碑。从零定义比修复受损形象更容易。

### 0.4 当前用户旅程

一个化工专业山西籍学生投递鹏飞的全流程：

```
认知阶段 → 搜索「鹏飞集团」→ 小红书/B站/牛客结果为零
搜索阶段 → 唯一信息源是 PDF 招聘简章，文字密集，无视觉设计
投递阶段 → 邮件发送，无自动确认，不知是否被收到
等待阶段 → 1 天、3 天、1 周，无反馈。心理：「石沉大海了」
结果阶段 → 大概率无回复（学生不会记得自己投过）；小概率面试通知（学生意外）
```

**每个触点都在发送同一个信号：这家公司不太在意校招这件事。** 这是 GreenSpark 在流程体验层最需要解决的基线问题。

### 1.2 关键竞品分析

**用友大易**：基于 YonGPT 大模型的全链路校招 SaaS。能做渠道（朋友圈精准广告、校园人才社区），做不了内容（不涉及短视频、B站/小红书、不做行业转译）。

**The Martec（澳大利亚）**：全球首个端到端 AI 雇主品牌平台。AI Story Hub 将员工回答转化为视频/博文/社交帖。不支持中国社媒生态。定价面向大型跨国企业。员工 UGC 驱动模式不适用能源企业员工。

**Phenom（美国）**：最先进的 AI 人才体验平台。超个性化网站、对话式 AI 求职助手。完全面向欧美市场，不支持中国社媒。

**因赛集团**：AIGC 能力最强（3 小时 1,380 张图像 + 231 条视频）。但通用营销工具，无校招场景模板，无学生分层。

**核心结论**：没有人做「行业认知→年轻化转译」这件事。这是 GreenSpark 填的空白。

### 1.3 跨行业案例

**玛氏**：[1]
- AI 内容管线：选题 → 文案 → 视觉全流程
- 小红书年涨粉 4万+、互动 10万+、直接转化 2万+ 人次
- 账号用打工人视角：「我有 15 天年假」而非「我们提供行业领先的假期福利」
- 对鹏飞的启示：**AI 降成本 + 真人人设 + 简介就开始做雇主品牌**

**字节跳动**：[2]
- 42% 校招生首选率
- 「心脏只有一个能跳动」的负面口碑没有劝退——因为学生收到的是完整信号包：高薪+快速成长+简历镀金+加班
- 对鹏飞的启示：**不用消除所有负面认知，正面信号的强度覆盖负面成本即可**

**蔚来/特斯拉**：[3]
- 入职 1 年新员工在小红书自发写真实工作记录，传播力远超官方内容
- 离职员工持续为品牌种草
- 对鹏飞的启示：**最有效的内容不是 HR 写的，是在职员工自己愿意讲的**

**宁德时代**：[4]
- 透明按校分级薪酬体系创造「规则明确」的认知
- 校园大使要求小红书/B站运营力
- 对鹏飞的启示：**校园大使是内容共创者，不是贴海报的**

**特变电工**：[5]
- 年轻化图漫+朋友圈精准广告百万级曝光
- 叙事从「企业有多大」转向「你能成为什么样的人」
- 对鹏飞的启示：鹏飞最直接的对标对象，共享传统制造业转型的约束和机会

## 二、信号类型学——什么样的校招内容有效？

对跨行业案例的归纳产生了一个分析框架：

```
                    信号来源
                 个体/体验        组织/设计
            ┌────────────────┬────────────────┐
    高信任   │                │                │
    真实性   │ Type 1         │ Type 2         │
            │ 有机体验型       │ IP 锚定型       │
            │ 员工口碑/面经    │ M&M's / 企鹅    │
            │                │                │
            ├────────────────┼────────────────┤
    低信任   │                │                │
    工具性   │ Type 3         │ Type 4         │
            │ 平视叙事型       │ 官方宣告型       │
            │ 打工人视角       │ 企业宣传稿       │
            │                │                │
            └────────────────┴────────────────┘
```

**四个关键结论**：

1. **Type 1**：「字节和心脏只有一个能跳动」这句负面标签没劝退——因为正面信号包（高薪+成长+镀金）的总价值远超加班的代价。负面口碑的传播者没有利益动机，反而让正面信号更可信。

2. **Type 2**：玛氏用 M&M's 豆、腾讯用企鹅——消费者品牌好感直接迁移到雇主品牌。鹏飞没有消费级 IP，替代方案：把「氢」做视觉拟人化，让氢能重卡成为固定视觉锚点。

3. **Type 3**：玛氏小红书的简介从打工人视角写——「我是一个有 15 天年假的打工人」而非「玛氏为您提供优厚福利」。福利是展示的不是宣告的。展示让人相信，宣告让人怀疑。

4. **Type 5（扩展）——求职流程体验**：应聘者亲身经历的投递/面试/宣讲会体验是最强品牌信号。好想你宣讲会送大枣礼包→学生自发帮它传播。48% 被拒者不知为何被拒[6]→模板式拒信损害未来投递意愿。鹏飞当前邮件投递+无确认的模式是品牌减分触点。

**对鹏飞的操作建议**：

| 类型 | 怎么做 |
|------|-------|
| Type 1 | 孵化入职 1 年新员工在小红书/牛客自发写工作记录，不美化不遮掩 |
| Type 2 | B站科普动画中「氢小球」IP 形象；氢能重卡作为固定视觉锚点 |
| Type 3 | 所有内容以打工人视角说话——「我是鹏飞制氢基地的工程师」而非「鹏飞集团为您介绍」 |
| Type 4 | 仅用于校招官网和 JD——且内容须与 Type 1-3 信号一致，不画饼 |
| 流程体验 | 投递→5 分钟自动确认；宣讲会现场扫码输出定制化职业想象；面试后 24h 给反馈 |

## 三、内容样例

### 3.1 短视频脚本（B站/抖音，60 秒）

**目标受众**：化工/材料专业大三学生
**策略**：用课程内容做 hook

| 时间 | 画面 | 文案 |
|------|------|------|
| 0-3s | 化工厂全景→快速切换到氢能实验室 | 「你在《化工原理》上学的精馏塔，正在造地球上最干净的能源。」 |
| 3-15s | 鹏飞氢能产业园航拍+电解水设备 | 「鹏飞在山西建了从制氢、储运到加氢的全产业链。学化工，不只可以去炼油厂。」 |
| 15-35s | 年轻工程师在实验室/团队讨论 | 「去年入职的张工，天大化工硕，在做电解水催化剂优化。他实验室的方向在这里变成了产业项目。」 |
| 35-50s | 健身房/食堂/宿舍 | 「薪酬对标一线城市，生活成本是太原水平。公积金最高比例，第一年住房补贴全覆盖。」 |
| 50-60s | 鹏飞 Logo + 二维码 | 「鹏飞 2026 秋招已启动。搜鹏飞氢能，看看未来的你做什么。」 |

### 3.2 小红书科普帖

**标题**：「说碳中和的那么多，氢能到底是什么？」🔥

**正文**：

> 化工🐶最近发现一家叫鹏飞的企业在山西搞氢能全产业链。
>
> 简单说：
> ☀️ 风光发电
> 💧 电解水制氢
> 🚛 氢能重卡跑零碳物流
> 📡 5G+AI 调度车队

> 实绩：
> ✅ 2 万吨/年制氢基地已投产
> ✅ 氢能重卡示范线已跑通
> ✅ 零碳智慧物流平台上线
> ✅ 数字化人才缺口大（不是只招挖煤的！！
>
> 秋招已开始，工科和计算机同学都可以看 👀

> #校招 #化工求职 #新能源 #氢能 #碳中和 #鹏飞集团

### 3.3 宣讲会互动文案

**场景**：线下宣讲会开场破冰

> 「先做个小调查——今天来之前，你们对鹏飞的第一印象是什么词？」
>
> （等待回答：煤炭、传统、山西……）
>
> 「记住你们现在说的词。45 分钟后，如果你们对鹏飞的印象没有任何变化，今天我白来了。但如果 45 分钟后你们发现，鹏飞在做的事情跟特斯拉、宁德时代、华为的某些业务在同一个赛道上——我希望收到你们的简历。」

## 五、团队能力

**项目负责人**：社会工作专业在读，正在求职中，投递人力资源管理方向。

**双重身份**：

- **作为社工专业学生**：方案中的优势视角、生态系统理论、案主自决不是文献综述的装饰——是三年训练内化的分析工具。方案之所以不用市场营销的「差异化定位」或管理学的「流程优化」作为切入点，是因为社会工作的根本关切是「人如何在环境中做出自主、知情的决定」。
- **作为正在求职的应届生**：方案中关于候选人体察、求职方向迷茫、宣讲会体验触发分享的论述，大量来自第一手求职经历。2026 年秋招季，我在投递 HR 方向岗位的过程中同步观察了鹏飞、宁德时代、比亚迪、玛氏、字节跳动的雇主品牌传播实践。这套方案里的洞察来自求职者视角的真实体感。

**相关能力**：AI 应用（大模型 API、Prompt Engineering）、雇主品牌研究（求职过程中系统观察 10+ 家企业校招传播）、内容运营理解（小红书/B站/牛客校招生态的直接使用体验）、高校触达经验（应届生第一视角参与招聘会、宣讲会、高校就业群）。

# 一、 命题前置分析
### 1.1 基础与核心现状

通过查阅鹏飞 2026 年校园招聘简章，各高校就业网公开信息及第三方求职平台检索后发现，**鹏飞校招的核心问题并非传播效果不佳，而是几乎不存在相关传播**。学生在小红书搜索鹏飞集团相关内容无任何结果。
当前鹏飞校招在薪资，地域，招聘规模层面均无竞争优势，但手握氢能链主的实体产业资产，该**资产从未被转化为面向求职学生的传播内容**。零传播存在感比负面口碑的影响更严重，学生未投递鹏飞并非经过对比后主动放弃，而是在整个求职过程中，鹏飞从未进入其信息获取视野。

| 维度      | 真实状态                                                  |
| ------- | ----------------------------------------------------- |
| 社交媒体存在感 | 在 B站、小红书、牛客搜索「鹏飞集团 校招」，结果为零                           |
| 投递方式    | 仅支持邮箱投递，无自动确认回复。学生不知道简历是否被收到                          |
| 宣讲会覆盖   | 专场宣讲集中在山西省内高校（山西能源学院、山西财经大学、太原理工大学）。省外 985/211 无专场    |
| 薪资      | 6,000-9,000 元/月。宁德时代 985 硕年包 23.6-33W，比亚迪 985 硕底薪 12K |
| 工作地点    | 总部山西孝义（县级市），另有晋城、长治、阳泉、天津、十堰                          |
| 招聘规模    | 约 50 人（2026 届）                                        |
| 校招项目    | 鸿鹏生（普通校招，本科+）+ 麒鹏生（管培，985/211本科，硕士须第一学历 985/211）      |
| 核心资产    | 山西省氢能产业链链主、2 万吨/年制氢基地已投产、250kW 氢燃料电池、氢能重卡已上路          |
## 1.2 能源行业校招格局定位

鹏飞的校招竞争并非面向全行业所有企业，而是面向学生心智中能源行业标签下的固定选项，不同梯队企业在规模，薪资，认知标签层面差异显著。


| 梯队 | 代表企业 | 规模 | 薪资参考 | 鹏飞 vs 它 |
|------|---------|:--:|------|------|
| 央企巨头 | 国家能源、中石化、中石油 | 万人级 | 硕士 15-25W | 拼不了稳定，拼可见度 |
| 新能源热门 | 宁德时代、比亚迪、隆基 | 数千人级 | 硕士 18-33W | 拼不了薪资，拼成长速度和被看见速度 |
| 区域强企 | 特变电工 | ~1,700 人 | 未公开 | 最直接的对标对象：同是传统转型、同在非一线、同在尝试年轻化传播 |
| **鹏飞当前** | — | **50 人** | **6-9K** | 零社媒认知。但氢能链主身份是实打实的产业资产 |


鹏飞的核心竞争策略，不在薪资维度与宁德时代，比亚迪正面竞争，不在稳定性维度与央企正面竞争，**鹏飞的核心竞争维度是「被看见的速度」和「氢能赛道的稀缺性」**。化工类毕业生进入比亚迪的数万人团队中只是普通一员，进入鹏飞的 50 人校招团队中即可成为氢能产业的核心定义者，该叙事逻辑目前尚未在传播环节被激活。

## 1.3 跨行业可迁移经验参考

鹏飞无需参考能源行业的现有校招传播实践，该领域暂无成熟的标杆案例，可借鉴快消，互联网，新能源汽车领域的底层传播机制，以下五个案例各自贡献了一个可迁移的方法论：

| 玩家 | 鹏飞该学什么 |
|------|------------|
| 玛氏中国 | 小红书用「打工人」视角说话，不居高临下。简介就开始做雇主品牌——「我是有 15 天年假的玛氏员工」而非「玛氏提供行业领先的假期福利」 |
| 字节跳动 | 「心脏只有一个能跳动」的负面标签没劝退学生，因为学生收到的是完整信号包（高薪+成长+镀金，代价是加班）。正面项总价值覆盖了负面成本，且负面口碑的传播者是没有利益动机的员工，反而建立了「他们说的话可信」的认知基础 |
| 蔚来/特斯拉 | 入职一年的新员工在小红书自发写工作记录，离职员工持续为品牌种草。最有效的校招内容不是 HR 写的 |
| 宁德时代 | 透明按校分级薪酬体系创造了「规则明确」的信号——学生自发在牛客讨论，形成了免费的社会验证。校园大使明确要求小红书/B站运营力 |
| 特变电工 | 年轻化图漫+朋友圈精准广告达成百万级曝光。叙事从「企业有多大」转向「你能成为什么样的人」 |


上述五个案例指向同一结论，**成功的校招传播的共同特征是让企业外部的主体主动为企业发声**。玛氏依托普通员工的第一视角内容，字节依托员工自发吐槽中夹带的正向信息，蔚来依托离职员工的持续正向宣传。鹏飞当前的状态是没有任何外部主体主动为其发声，既没有在职员工在小红书发布工作体验，也没有求职学生在牛客讨论面试经验，甚至不存在校招简章之外的可触达内容。GreenSpark 的核心任务并非直接替鹏飞发声，而是为鹏飞的在职员工，离职员工，新入职人员提供对应的工具，素材，激发其主动为品牌发声的意愿。

# 二、解决方案
### 2.1 方案概述

**项目名称：GreenSpark·绿能引力**
依托 AI 将鹏飞的氢能，零碳物流和 5G 数智化业务转化为适配不同专业学生的内容，在 B 站，小红书和高校社群中精准分发。核心理念是将鹏飞的内容产能从零到一升级，实现从学生搜不到到能搜到，能看懂，想转发的转变。

方案运转逻辑分为四步，对应四层技术架构，依次为行业知识图谱将业务信号转译为不同专业学生可理解的叙事，AIGC 批量产出短视频脚本，宣讲文案和科普内容，按高校层次，专业集群，年级节点，信息渠道四个维度精准分发，由校园大使和员工 KOC 作为信源扩散，回收数据完成迭代。

### 2.2 系统架构与功能模块

本部分通过总览说明 GreenSpark 针对的核心问题，落地路径与产出结果，五个模块分别对应鹏飞校招的一项具体痛点。

| 模块              | 解决的痛点                                | 关键输出物                          |
| --------------- | ------------------------------------ | ------------------------------ |
| EVP 信号转译引擎      | 学生看不懂「氢能全产业链」跟他有什么关系，信号编码不兼容         | 专业-业务线匹配矩阵、分层叙事策略卡片            |
| AIGC 内容创作工厂     | 单个人力资源团队无法产出覆盖几十所高校的差异化物料，人工产能限制精准触达 | 短视频脚本、宣讲文案、小红书科普帖、校园大使素材包      |
| 三层生态分发引擎        | 「一套物料群发所有渠道」，内容与受众错配                 | 高校×专业×年级×渠道分发策略表               |
| 校园大使与 KOC 赋能工具包 | 缺少人员在学生社群中发声，官方声音穿透力弱于同辈推荐           | 话术库、FAQ、一键转发素材、员工故事采集模板        |
| 效果追踪与策略迭代看板     | 无法判断内容实际效果——缺乏从曝光到投递的归因链路            | 渠道→投递转化归因表、认知扭转率追踪、内容 A/B 对照报告 |

**五个模块的调用关系为**，EVP 信号转译是所有下游模块的认知底座，内容工厂调用转译结果生成各平台素材，分发引擎调用内容工厂的产出完成渠道匹配，大使工具包调用内容工厂的物料完成格式适配，追踪看板收集所有下游数据反向校准转译规则和内容策略，形成完整闭环，并非五个独立的功能点，而是一条持续运转的生产线。

### 2.3 核心创新

**三项创新并非优化类的锦上添花内容，而是解决鹏飞校招三个从零到一的基线问题。**

（1）**创新一：EVP 信号转译，实现从学生看不懂到能看懂的转变**。现有雇主品牌产品仅覆盖内容的承载和分发环节，未涉及认知翻译。GreenSpark 将鹏飞的每项业务映射到不同专业的课程语言，面向化工学生讲解电解水制氢的膜电极，关联《化工原理》中的传质和催化内容，面向计算机学生讲解零碳物流的实时调度，关联《算法导论》中的 NP-hard 问题。所有映射均可溯源，以 2 万吨/年制氢已投产这类硬数据为支撑，不使用空泛形容词，该映射机制不以创意写作为目标，以事实可验证为底线。

（2）**创新二：三层分发，实现从推送信息到激活生态的转变**。传统校招传播仅在个体层面发力，若学生的室友传递山西没有优质就业机会的观点，导师未提及鹏飞，学院就业网仅推送互联网和快消类企业，即便内容精准触达学生，也难以抵消三层环境带来的反向拉力。GreenSpark 在微观层实现让学生感知到内容是专门面向自己的，在中观层通过校园大使和员工 KOC 营造身边有人投递鹏飞的社会验证，在宏观层于 B 站知识区推出《氢能产业 101》科普系列，在小红书搭建氢能求职相关话题，推动鹏飞进入学生的公共信息叙事体系。

（3）**创新三：求职方向诊断前置，实现从向学生宣传企业优势到帮助学生明确自身适配方向的转变**。2025 届校招调研显示，超 60% 的应届生不清楚自身适配的岗位类型。GreenSpark 在学生输入学校和专业信息后，不会直接推送鹏飞的企业介绍，而是先展示五个该专业可进入的行业方向供学生选择偏好，再基于选择生成定制化内容。若学生选择的是非鹏飞覆盖的方向，系统将给出中立建议，不会强行引导学生投递。即便学生本次未投递，在未来 3 到 5 年的校招窗口期内仍有可能关注鹏飞。

**三项创新指向同一设计原则，即 GreenSpark 并非说服引擎，而是理解引擎**。它不会代替鹏飞说服学生投递，而是帮助学生理清三件事，分别是鹏飞正在开展的业务内容，自身所学知识在鹏飞的应用场景，以及自身是否真正适配鹏飞的岗位。说服会引发受众的防御心理，理解则会建立信任，这是本方案与其他校招工具最核心的差异。

### 2.4 具体打法

**鹏飞不用在每个方面都赢。它需要在四个选定的打法上做到位。**
>打法一信号编码决定打法二素材采集的采集方向和采集对象选取标准。打法二的素材供给决定打法三分发是否有充足的优质内容可发布。打法三的渠道数据反馈决定打法四节奏下一季度的策略调整。四大打法并非可选的菜单选项，而是一套需要同时启动、相互校准的运营系统。

（1）**打法一： EVP 分层信号编码**。面向不同专业的学生传递适配的企业内容，面向化工学生讲解催化剂相关内容，面向计算机学生讲解调度算法相关内容，面向经管学生讲解碳交易相关内容。所有内容的编码规则统一，确保业务事实可溯源，不编造，不夸大，不画饼。当前鹏飞仅有的传播物料为统一版本的 PDF 招聘简章，所有专业的学生获取的内容完全一致。GreenSpark 的第一优先级产出就是替代这份统一版本的简章，为五个核心专业方向分别产出一套差异化物料。

（2）**打法二：真人素材驱动的内容生产**。AI 可完成文本生成工作，但无法产出真实的在岗体验。GreenSpark 在内容工厂的上游设置两个轻量化采集通道，一是由在职员工提交一条公司最值得对外宣传的内容，形式为三句话搭配一张图，由 AI 转译为各平台发布的初稿，二是由离职校友在校友通讯中回应你会如何向学弟学妹推荐鹏飞，经本人授权后转化为校园大使素材包中的社交推荐语。AI 仅负责转译和格式化工作，不会编造事实，所有对外发布的内容均可追溯到采集记录中的具体人员和具体事件。

（3）**打法三：三层生态分发**。将三层干预分别对应到渠道，发布主体和预期效果。三层分发属于协同关系而非并列关系。微观层负责首次认知接触，即让受众第一次知晓鹏飞。中观层负责态度转化，即传递真实参与人员的正向反馈。宏观层负责长期认知锚定，即建立鹏飞等于氢能的品牌关联。缺少任意一层，其余两层的投入回报都会被稀释。

| 生态层    | 渠道                      | 谁来发           | 一个例子                            | 预期效果                   |
| ------ | ----------------------- | ------------- | ------------------------------- | ---------------------- |
| 微观（个体） | 牛客私信、高校就业网精准推送          | 系统自动分发        | 化工专业学生在牛客收到「你的专业在鹏飞能做什么」        | 「这是给我的」——精准感知          |
| 中观（同辈） | 小红书/B站评论区、高校就业群、校园大使朋友圈 | 校园大使 + 员工 KOC | 入职一年的太理机械硕士在小红书发「在鹏飞做氢能装备的这一年」  | 「我身边有人去了」——社会验证        |
| 宏观（场域） | B站知识区、小红书话题、知乎圆桌        | 官方号 + 行业 KOL  | B站系列《氢能产业 101》专题片、小红书#氢能求职 话题运营 | 「这家公司在更大的叙事中有位置」——价值认同 |

（4）**打法四：常态化内容节奏**：当前鹏飞及多数传统企业的校招传播为脉冲式，仅在招聘季的2个月内进行密集推广，其余10个月均处于静默状态。但认知偏见的修正需要多次、正向、一致性信号的累积，因此将全年划分为四个运营波段。

|阶段|时间|运营目标|内容策略|考核指标|
|---|---|---|---|---|
|非招聘季|1-2月、5-8月|认知修正与兴趣种草|在B站发布氢能、零碳科普内容；在小红书发布员工日常vlog；在知乎发布行业洞见|阅读量、互动率、正面评论占比|
|预热期|8-9月|意向培育与简历预激活|发布岗位解读、笔试面试经验帖、组织学长学姐答疑|校招官网UV、社群进群量|
|招聘期|9-11月|申请转化与口碑扩散|发布宣讲会互动文案、投递截止倒计时内容、offer holder分享内容|投递量、笔试通过率、面试到场率|
|复盘期|12月|数据沉淀与策略迭代|分析各渠道、各内容形式的转化数据，更新学生画像和编码规则|下一年度策略手册更新版本|





# 三、落地路径与预期效果

## 3.1 落地路径
不制定长期三年规划，优先落地3个月可跑通的MVP最小可行产品，后续基于数据反馈确定下一步动作。

|阶段|时长|动作|产出|验证假设|
|---|---|---|---|---|
|试点阶段|1-2周|通过LLM API注入行业知识生成Prompt；产出首批差异化物料；在3所省内目标高校试点分发|3套短视频脚本、2套宣讲文案、5篇小红书科普内容|分层内容的互动率比统一物料高3倍以上|
|搭建阶段|1-2月|使用飞书多维表格管理内容资产；通过飞书搭建自建投递表单替代当前纯邮件模式；启动员工故事采集和校园大使招募|半自动化内容生产管线v1.0；投递到确认到追踪的全流程体验闭环|投递体验改善可独立提升投递完成率|
|扩展阶段|3个月|完成5-8所目标高校的深度渗透；积累一整季的内容、渠道、转化相关数据，建立基准表|可复用的校招传播策略手册v0.1|可实现从0可搜索内容到20条以上可搜索内容的增量|

## 3.2 效果指标
效果指标不与行业平均水平对标，因当前业务基线为零，全部指标均与自身基线对标。

|指标|当前基线|3个月目标|
|---|---|---|
|小红书、B站可搜索内容|0条|20条以上|
|投递体验|邮件无确认|飞书表单 + 自动确认 + 48h状态提醒|
|内容产能|0套学生向物料|1天产出3套差异化物料|
|目标高校内容渗透|覆盖3-5所省内宣讲高校|覆盖5-8所高校，每所高校设置不少于2个内容触点|
|校园大使|未启用|配置3-5人，为每位大使配备专属素材包|

上述目标数值设置偏保守，因鹏飞当前相关业务的起点为零。从零条可搜索内容拓展至20条以上，从零套学生向物料实现单日产出3套，从零名校园大使配置3-5名大使，**属于从零到一的验证期**，并非从十到百的放量阶段。**3个月后形成的基准表将直接决定第二年秋招的策略预算和资源配比**，明确哪些内容形式跑通、哪些渠道转化率显著更高、哪些高校的渗透成本最低，这些均为第一阶段数据需要明确回答的核心问题。
